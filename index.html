Cows & Bulls

Guess the hidden number and use the hints after each try to crack the code.

💖

Room Setup

Create a room, send the code to your friend, and have them join from their own device.

Your number never appears in the UI unless you reveal it or inspect the database directly.

Rules

  1. The number can never start with 0.
  2. You cannot repeat digits.
  3. Cows = number of digits correct overall.
  4. Bulls = number of digits in the correct place.
  5. Both players set a secret, then turns alternate automatically.

Supabase setup

Before this works online, paste your Supabase URL and anon public key into the script, then make sure the cowsbulls_rooms table and realtime are enabled in Supabase.
